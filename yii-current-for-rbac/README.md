# yii current for rbac 这是noecs的通用权限系统。
> 使用Yii框架内置rbac权限管理。
> 更简洁的代码，有利于二次开发。
> 通用性强，使用方便。

###### 作者名称：noecs
###### 作者网址：https://soera.cn
###### 作者邮箱：noecs@qq.com

### 安装说明
1.解析域名。
2.创建虚拟主机，目录指向本程序的 /backend/web/。
3.导入程序的数据库。
4.配置/common/config/main-local.php的数据库链接。
（注意：如果是lnmp环境，需要配置.user.ini文件，添加项目根目录权限即可）
5.访问域名，账号admin,密码admin。
